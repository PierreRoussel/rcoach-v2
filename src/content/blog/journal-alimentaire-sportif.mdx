# Journal alimentaire sportif : méthode simple

Le **journal alimentaire** n’est pas réservé aux compétiteurs. Pour un sportif amateur, c’est un outil de lucidité : il montre ce que vous mangez réellement, pas ce que vous pensez manger.

## Ce qu’un bon journal doit contenir

- Les **repas** et collations avec quantités réalistes
- Les **macros** ou calories selon votre niveau de détail
- Le **contexte** : journée d’entraînement, repos, sortie

## Quand le remplir

Le plus fiable : **au moment du repas** ou juste après. Reporter la fin de journée augmente les oublis, surtout sur les grignotages.

## Erreurs fréquentes

- Sous-estimer les huiles, sauces et boissons
- Oublier les week-ends
- Changer tous les paramètres chaque semaine sans laisser le temps d’observer
